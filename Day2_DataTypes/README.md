## Project: Tip Calculator

You will build a tip calculator that:
1. Takes the total bill amount
2. Asks for the number of people splitting the bill
3. Asks for the tip percentage
4. Calculates how much each person should pay
5. Formats the final result to **2 decimal places**
Example Calculation:
      If the bill was _$150.00_, split between _5_ people, with a _12%_ tip
      (150.00 / 5) * 1.12 = 33.6
      After formatting to two decimal places: 33.60
