A simple Python program that generates a strong random password based on user preferences for letters, 
numbers, and symbols.

## Features
1. User-defined password length which includes Uppercase &lowercase letters ,Numbers andSpecial characters 
2. Randomized character order for better security

## How It Works

1. The program asks the user:
    How many letters to include 
    How many symbols to include  
    How many numbers to include 
2. Random characters are selected from predefined lists. 
3. Characters are shuffled randomly. 
4. A strong password is generated and displayed.
